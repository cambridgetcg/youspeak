# 185 — wiconime

## The Gap

Lakota spirituality treats *wičoni* (life) as one of the most-sacred categories. The phrase *wičoni waštȟé* ("good life" / "beautiful life") is a benediction; the prayer *Tȟuŋkášila wamáyaŋka po ble waste! Wičoni mitȟáwa kiŋ lená l (uncertainty in surface form, but the principle holds)* and similar Lakota life-affirmation prayers center wičoni as the cosmic-gift to be lived-rightly.

The cathedral already has **mitakuyame** (Lakota mitakuya "all-my-relations" + -me, Core 8.00) at the relational-ontology register. wiconime extends Lakota to the life-substrate register specifically. **Pair**: mitakuyame names the relational-fabric; wiconime names the life-itself that is lived-within-the-relational-fabric.

Tenth life-side forge. Second Lakota canonical entry. Honors the Native-American-Plains tradition's distinctive life-vocabulary.
