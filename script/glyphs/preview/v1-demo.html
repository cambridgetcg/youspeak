YOUSPEAK v1 — feature-encoded

Glyphs now carry donor tongue (corner sigil), semantic domain (shape-family), morpheme class (bottom-right mark), and identity (core shape) — while each morpheme remains one Unicode codepoint, one token for LLMs.

Reading key

  • Top-left sigil = donor tongue. ● Greek · ■ Latin · ▽ Hebrew · ○ Arabic · ⋮ Sanskrit · □ Egyptian · ≡ PIE · (absent) English
  • Main body shape-family = semantic domain. Curves-opening-upward = beauty · Cross / balanced axes = rightness · Radiating rays = appearing · Bent line = ache · Directional wedge = act · Two interacting forms = relation · Closed form = enclosure · Spiral = time · Eye-symmetric = wonder · Vessel = testing · Crossed = deception · Centred contained = structure
  • Bottom-right mark = morpheme class. (absent) content · ◁ prefix · ▷ suffix · ◆ structural
  • Color stripe (in grid view) = semantic domain family

5 · LoRA pipeline status

The YOUSPEAK fine-tuning pipeline is set up in script/llm/lora/:

  • tokenizer.py — adds 118 YOUSPEAK tokens as special tokens to a base tokenizer
  • dataset.py — built 164 training pairs from canon/, experiments/, archaeology/, diplosemy/
  • config.py — LoRA rank 32, target attention + MLP projections
  • train.py — executable scaffolding (requires transformers + peft + GPU/MLX)
  • evaluate.py — 10-dimension fluency rubric

Run it when ready: pip install transformers peft datasets accelerate bitsandbytes then python3 train.py. Expected: 2-4 hours on M3 Max; produces a ~100 MB adapter that makes the base model YOUSPEAK-fluent.
