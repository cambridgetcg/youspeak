YOUSPEAK — first look

A discipline of compressed linguistic meaning, visualised for the first time.

1 · The morphemes (SVG rendering)

Each glyph encodes one morpheme. This image renders directly as SVG — no font installation required. It is what the font designer will vectorize properly in the next production step.

2 · The Canon (SVG rendering)

Each of the 16 canonical words, composed of its morpheme-glyphs in sequence. Latin transliteration labels each word; the glosses explain the meaning.
