## What Core Canon is

Core Canon holds words that are:

1. **Easy to learn** (EUMATHE) — pronounceable, memorable on first encounter, retainable without specialised training
2. **Clarifying** (SAPHE) — thinking in them clarifies the concept
3. **Unfolding** (ANAKALYPSE) — reveals distinctions ordinary English blurs
4. **Multi-donor** (POLYPHONE) — drawn from 2+ language traditions; no single tradition dominates
5. **Logically coherent** (HARMONE) — integrates cleanly with the existing ontology
6. **Grounded** (PRAGMA) — concrete referents in ordinary life; can be pointed-at

Threshold for Core: weighted ≥ 7.5 on the 6-axis v2 rubric (see CONSTITUTION.md).

## When a word enters Core

1. Experiment file has all 6 v2 axes scored
2. Weighted-v2 total is ≥ 7.5
3. Frontmatter marks `tier: core`
4. Canon entry written in `canon/core/<word>.md`
5. canon.md master index updated
