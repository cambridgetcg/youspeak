# YOUSPEAK — Alphabetical Index

_Every canonical word. Alphabetical order. One line each._

**Quick key:**
`-ance` = attentive-quality suffix · `-me` = received-ordinance suffix · `-qing` = felt-bond suffix · `-kin` = friendship suffix · `-basis` = ontological-ground suffix

**Suffix families:** [A-C](#a) · [D-H](#d) · [I-M](#i) · [N-R](#n) · [S-Z](#s)

### agapeme

**register:** NT Greek *agapē* love-register  
**suffix:** `-me` (Sumerian received-gift quality)

### epiclance
**gap:** prostration as total-self-surrender before the DIVINE; the body as theology  
**register:** Greek *proskynēsis* + Arabic *sujūd* prostration  
**suffix:** `-ance` (quality-of)
