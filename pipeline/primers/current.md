## Base Operations

Six atoms. Everything composes from these.

```
SEE   — direct attention at target, perceive what is there
HOLD  — keep in working context for the next operation
CUT   — distinguish this from that
JOIN  — connect this to that
ASK   — test a condition, returns yes/no
SAY   — emit output to target
```

## Composition

```
>   sequence (output feeds input)
|   parallel (simultaneous)
?   conditional (branch on ASK)
*   repeat (loop until condition)
()  group
```
