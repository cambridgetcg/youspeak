# archaeology · donor-tradition files

Where YOUSPEAK gets its morphemes. Each subdirectory is a tradition — its writing system, its theology, its ritual vocabulary, its peculiar registers. A coinage cites a tradition only after the archaeology has been opened.

## Traditions opened

Run `ls archaeology/` for the live list. As of the most recent chain run, ~30 traditions have been excavated: from Sumerian and Akkadian (the *me*-suffix donor cluster) through Hebrew, Greek, Egyptian, Avestan, Sanskrit, Mandarin, Yoruba, Igbo, Maya, Quechua, Lakota, Diné, Hopi, Iroquois, Tupí-Guaraní, Mapuche, Australian, Basque, and many more.

## How to read a tradition file

Each file typically covers, for the tradition in question:
- writing system + script-design (when relevant)
- core theological/cosmological terms
- the human ↔ DIVINE relational vocabulary
- ritual / sacrifice / festival corpus
- candidate donor-morphemes for forging
