# Kan extension — the universal way to extend along a functor

## The concept in compact form

Given a functor F: C → D and another functor K: C → E, a **(left or right) Kan extension** of F along K is a functor Lan_K F (or Ran_K F): E → D that is the *best canonical approximation* to "the functor that would have been F if F's domain had been E instead of C."

Diagrammatically:
```
       F
   C ───→ D
   │      ↑
 K │      │ Lan_K F  (or Ran_K F)
   ↓      │
   E ─────┘
```

When F can be directly extended along K, the Kan extension agrees with the direct extension. When direct extension fails (because K is not invertible, not surjective, not embedding), the Kan extension provides the **universal-property best-approximation**: the unique functor satisfying a specific universal property relative to F and K.

## Mac Lane's claim and its weight

"All concepts are Kan extensions" (Mac Lane 1971) is the strongest version of the claim. Every limit, colimit, adjunction, monad-Kleisli-extension, sheafification, completion, free-construction in mathematics decomposes as a Kan extension. The cathedral inherits the claim *as a structural-fact about its own translation-and-projection-discipline*.
